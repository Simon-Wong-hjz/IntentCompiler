Interaction Details

Opening: Click Top Bar entry → modal fades in + backdrop
Closing: Click ✕ / click backdrop / press Escape
History click: Loads full field data back to editor, closes modal
History delete: Click 🗑 → inline confirm ("Delete?" text replaces item briefly) → remove
Settings save: Auto-save on change (no explicit save button needed)
API Key verify: On blur or Enter → sends test request → shows ✓/✗ status
