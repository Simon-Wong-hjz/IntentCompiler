Label order: FIELD NAME ? → operation hint

Why this order: The ? explains the field (what it is, when to use it), so it belongs next to the field name. The operation hint explains the input type (how to interact), so it sits separately after.

What is this?
Categorize your question so AI knows how to structure the answer.
Options explained:
• factual — verifiable facts, data, definitions
• conceptual — explain principles, how things work
• how-to — step-by-step instructions
• opinion — subjective assessment, recommendations
Example:
"What are React hooks?" → factual
"Why use hooks over classes?" → conceptual
conceptual factual how-to opinion

↓ Same checkbox with help expanded:
Allow AI to add fields ?
What does this do?
By default, AI Fill only populates fields already shown in the editor. When this option is checked, AI can also discover and add relevant optional fields it thinks are useful for your intent.
Example:
If you type "Write a technical blog post about WebSocket" with this checked, AI might add "tone", "target_length", and "structure" — even if you hadn't added them yourself.
Note:
You can always review and remove any AI-added fields before compiling.
