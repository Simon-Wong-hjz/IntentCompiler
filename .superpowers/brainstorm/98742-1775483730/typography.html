A. Inter + System Chinese

Modern UI standard. Inter handles all Latin text with excellent readability. Chinese falls back to each OS's best: PingFang SC (Mac), Microsoft YaHei (Windows). Slight visual mismatch between scripts but both are high quality. Most popular choice for bilingual web apps.

C. Plus Jakarta Sans + System Chinese

More personality. Geometric, rounded letterforms that match the friendly editor vibe and the bold Sunflower+Ink brand. Heavier weights (800) look great for headings. Chinese falls back to system fonts. Slightly more character mismatch but stronger brand identity.

Note on Chinese Web Fonts

CJK web fonts are 10-20MB per weight — not practical to load. All options use system Chinese fonts (PingFang SC on Mac, Microsoft YaHei on Windows, Noto Sans SC on Linux). The choice here is really about the Latin/English font and how well it pairs with the system Chinese font.
