# 13. 里程碑 / Phased Roadmap

## Phase 0：定义与验证（2-4 周）

目标：验证任务类型、字段最小集、renderer 是否能成立。

交付：

- 任务类型 taxonomy v0
- Prompt IR v0
- 6-8 个核心模板
- 纸面原型 / 低保真交互稿
- 少量用户访谈与可用性验证

## Phase 1：MVP（6-10 周）

目标：跑通无 AI 主流程。

交付：

- 模板选择
- 表单构建
- IR 编译
- 实时渲染
- 复制 / 保存模板
- 基础埋点

上线标准：

- 30-60 秒内可完成一次构建
- 主要模板的复制率达到可接受水平

## Phase 2：AI Assist 插件（4-6 周）

目标：补充自然语言到草案的快捷入口。

交付：

- 原始问题转 IR 草案
- 低置信度题型识别
- 0-2 个追问策略
- AI 填充字段高亮

上线标准：

- AI 草案接受率显著高于人工从零填写
- 追问率保持低位

## Phase 3：模板资产层（6-8 周）

目标：把单次构建转化为长期资产。

交付：

- 模板版本管理
- 模板 fork
- 团队共享
- 模板分析仪表盘

## Phase 4：生态扩展（后续）

- 模型适配 renderer
- API / SDK
- 团队权限
- 模板 marketplace
- 外部编辑器 / AI 平台集成
