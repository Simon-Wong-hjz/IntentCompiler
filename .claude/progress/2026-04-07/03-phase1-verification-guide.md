# Phase 1 Manual Verification Guide

**Date**: 2026-04-07
**How to run**: `npm run dev` then open the localhost URL in browser

## Phase 1 功能验证清单（中文优先 UI 版本）

### 1. 页面布局
- [ ] 固定顶部栏，显示 "Intent Compiler" logo（Plus Jakarta Sans 字体，粗体）
- [ ] 右侧有"历史"/"设置"按钮（灰色禁用状态）和 **中/EN** 语言切换（中 为激活态深色背景）
- [ ] 下方是 6 张任务类型卡片：**提问、创作、转化、分析、构思、执行**（中文）
- [ ] 卡片副标题为中文心智模型（如"我想知道一件事"）
- [ ] 再下方是 50/50 左右分栏（编辑器 | 预览），中间有细灰色分割线
- [ ] 最小宽度 1024px（缩小窗口时出现水平滚动条）

### 2. 任务类型选择
- [ ] 点击任意卡片 → 深色背景 + 金色文字（选中态）
- [ ] 其他卡片保持白色背景 + 浅金边框
- [ ] 悬停时有浅黄色背景过渡
- [ ] 宽屏 6 列一行，窄屏 (< 1280px) 3 列两行

### 3. 提问（Ask）任务类型编辑器
- [ ] 意图字段（空态）：**红色 2px 边框 + 红色阴影光晕**，48px 最小高度，placeholder "你想完成什么？"
- [ ] 意图字段（填入内容 + 聚焦）：金色 2px 边框 + 金色阴影光晕
- [ ] 意图字段（填入内容 + 失焦）：默认灰色边框，无光晕
- [ ] 下方 6 个字段按顺序，标签为中文：**背景、需求、约束、输出格式、问题类型、受众**
- [ ] textarea 输入随内容自动扩展高度
- [ ] 标签全部大写、灰色、小号字体、字间距加宽
- [ ] 输入框聚焦时边框变为金色
- [ ] placeholder 为中文（"自由输入"、"每行输入一项"等）

### 4. 实时 Markdown 预览
- [ ] 在任何字段输入内容 → 右侧预览区即时更新
- [ ] 预览用等宽字体（SF Mono / Cascadia Code / Consolas）
- [ ] 每个字段渲染为 `# 字段名` + 内容
- [ ] 多个字段之间用空行分隔
- [ ] 空字段不出现在预览中
- [ ] 字段顺序与定义顺序一致（Intent 永远第一）

### 5. 复制功能
- [ ] 意图非空 + 有内容时"复制到剪贴板"按钮为金色可点击
- [ ] **意图为空时按钮始终禁用**（即使其他字段有内容）
- [ ] 点击后显示"✓ 已复制！"约 1.5 秒后恢复
- [ ] 无内容时按钮为半透明禁用状态
- [ ] 复制失败显示红色"✗ 复制失败"

### 6. 空状态 & 任务切换
- [ ] 未选任务 → 编辑器显示"请先在上方选择一个任务类型"
- [ ] 点击 创作/转化/分析/构思/执行 → 编辑器显示 🚧"即将推出"
- [ ] **切换任务类型时，意图字段值保留**
- [ ] 如果其他字段有内容，弹出确认对话框"切换任务类型将清空除'意图'之外的所有字段，是否继续？"
- [ ] 点击取消 → 不切换，留在当前任务类型
- [ ] 点击确认 → 切换，清空非意图字段，保留意图
- [ ] 预览区无内容时显示"选择任务类型并填写字段以查看预览"

### 7. 视觉风格
- [ ] 奶油色页面背景（#fffdf5）
- [ ] Plus Jakarta Sans 字体（几何圆润的无衬线体，与系统默认字体明显不同）
- [ ] 整体 "Sunflower + Ink" 品牌感：金黄 + 奶白 + 深黑
- [ ] 左右面板可独立滚动（当内容超出时）
